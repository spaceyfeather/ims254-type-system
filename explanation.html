Formal

Headline/Subhead: Roboto Condensed
Intro/Body: Andada Pro
Quote: Caveat
I first chose the body text to be a sans-serif so that it was easier to read, while also having it look a bit more modern. The quote text is the least obtrusive script font I could find that still has that handwritten look. I went through a number of fonts for the header but ended up with the one that looked the most "official." Still looking for a better header font, though.
This system is meant to be simple and unobtrusive. Personally, there is enough contrast to the sections, without one completely overpowering the other. It makes it easier to focus.
